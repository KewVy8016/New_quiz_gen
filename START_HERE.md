# 🚀 เริ่มต้นใช้งาน QuizFlow

## วิธีรันโปรเจค

### แบบที่ 1: รันด้วย Server (แนะนำ) ✅

```bash
npm start
```

จากนั้นเปิด: **http://localhost:3000**

**ข้อดี:**
- ข้อสอบที่อัปโหลดจะถูกบันทึกเป็นไฟล์จริงใน `json/`
- ไม่หายแม้ปิดเบราว์เซอร์หรือ clear cache

### แบบที่ 2: เปิดไฟล์โดยตรง

เปิดไฟล์ `index.html` ในเบราว์เซอร์

**ข้อเสีย:**
- ข้อสอบจะเก็บใน localStorage เท่านั้น
- อาจหายถ้า clear browser data

---

## ทดสอบการอัปโหลด

1. รัน `npm start`
2. เปิด http://localhost:3000
3. คลิก "Upload New Quiz"
4. กรอกชื่อข้อสอบ (เช่น "ข้อสอบคณิตศาสตร์")
5. เลือกไฟล์ `sample-quiz.json`
6. ตรวจสอบว่าไฟล์ถูกสร้างใน `json/ข้อสอบคณิตศาสตร์.json`

---

## ฟีเจอร์ใหม่

✨ **ตั้งชื่อข้อสอบก่อนอัปโหลด**
- ระบบจะแสดง modal ให้กรอกชื่อ
- ชื่อเริ่มต้นมาจากชื่อไฟล์
- รองรับภาษาไทย
- กด Enter เพื่อยืนยัน, Escape เพื่อยกเลิก
