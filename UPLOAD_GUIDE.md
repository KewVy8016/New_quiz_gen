# คู่มือการอัปโหลดข้อสอบ

## วิธีการอัปโหลดข้อสอบ

### 1. รันเซิร์ฟเวอร์ Local (แนะนำ)

เพื่อให้ข้อสอบถูกบันทึกเป็นไฟล์จริงในโฟลเดอร์ `json/`:

```bash
npm start
```

จากนั้นเปิดเบราว์เซอร์ที่: `http://localhost:3000`

**ข้อดี:**
- ข้อสอบจะถูกบันทึกเป็นไฟล์ `.json` ในโฟลเดอร์ `json/`
- ไฟล์ `quiz-list.json` จะถูกอัปเดตอัตโนมัติ
- ข้อสอบจะไม่หายแม้ clear browser cache

### 2. เปิดไฟล์โดยตรง (ใช้ localStorage)

เปิดไฟล์ `index.html` โดยตรงในเบราว์เซอร์

**ข้อเสีย:**
- ข้อสอบจะถูกเก็บใน localStorage ของเบราว์เซอร์เท่านั้น
- ถ้า clear browser data ข้อสอบจะหาย
- ไม่มีไฟล์จริงในโฟลเดอร์ `json/`

## ขั้นตอนการอัปโหลด

1. คลิกปุ่ม "Upload New Quiz"
2. ระบบจะแสดง modal ให้กรอกชื่อข้อสอบ
3. ชื่อเริ่มต้นจะถูกสร้างจากชื่อไฟล์อัตโนมัติ
4. แก้ไขชื่อตามต้องการ (รองรับภาษาไทย)
5. กด "อัปโหลด" หรือกด Enter
6. เลือกไฟล์ `.json` ที่ต้องการอัปโหลด

## โครงสร้างไฟล์ที่ถูกสร้าง

เมื่อรันด้วย `npm start`:

```
json/
├── quiz-list.json          # รายการข้อสอบทั้งหมด (อัปเดตอัตโนมัติ)
├── ชื่อข้อสอบ-1.json       # ไฟล์ข้อสอบที่อัปโหลด
├── ชื่อข้อสอบ-2.json
└── ...
```

## การลบข้อสอบ

- ข้อสอบที่อัปโหลดจะมีปุ่ม "🗑️ Delete"
- คลิกเพื่อลบข้อสอบ
- **หมายเหตุ:** ถ้าใช้ localStorage จะลบเฉพาะใน browser, ถ้าใช้ server จะลบไฟล์จริง

## คำแนะนำ

✅ **แนะนำ:** ใช้ `npm start` เพื่อให้ข้อสอบถูกบันทึกเป็นไฟล์จริง

❌ **ไม่แนะนำ:** เปิดไฟล์ HTML โดยตรง (ข้อสอบจะอยู่ใน localStorage เท่านั้น)
